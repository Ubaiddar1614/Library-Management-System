
# 📚 Library Management System

Welcome to the **Library Management System** project! 🚀 A simple, yet powerful Java application designed using **Object-Oriented Programming (OOP)** principles to manage books, users, and core library functions. Whether you're a student, developer, or just a curious learner, this project showcases basic yet essential features for managing a library system.

---

## 🎯 Features

✨ **Key Features** of this Library Management System:

* **Add Books**: Easily add new books to the library collection.
* **Search Books**: Find books by title, author, or category.
* **Borrow & Return**: Handle book borrowing and returning with automated tracking.
* **Remove Books**: Remove outdated or unwanted books from the system.
* **Manage Users**: Handle library users (students, members, etc.) with personalized access.
* **View Records**: Access a comprehensive view of library book records and transactions.

---

## 🚀 Technologies Used

This project is built using:

* **Java** (The heart of the system)
* **Object-Oriented Programming** (OOP Concepts: Classes, Objects, Inheritance, Encapsulation)
* (Optional for future updates): **File Handling** for persistent data storage

## 🛠️ Getting Started

1. **Clone the repository**:

   ```bash
   git clone https://github.com/maheen-736/library-management-system.git
   cd library-management-system
   ```
2. **Compile and run the program**:

   ```bash
   javac src/*.java
   java src.Main
   ```
